## 第二章：机械学习基础
机械学习：大量数据 ---> 机械学习算法 ---> 模型（输入样例） ----> 输入结果   
### 数据集  
 特征空间（在多维度的位置）   
 分类任务就是在特征空间切分   

 ### 主要任务     
 监督学习：分类，回归
 分类任务：二分类，多分类，多标签
 回归任务：房屋价格，市场分析，学生成绩，股票价格   
 回归任务可以简化成分类任务   

 ### 机械学习方法分类  
监督，
非监督（降维度，聚类分析，方便可视化，异常检测），
半监督，
增强（AlphaGo，无人驾驶，机器人），
批量，
在线（及时反映新的环境变化，新数据不好），
参数，
非参数
